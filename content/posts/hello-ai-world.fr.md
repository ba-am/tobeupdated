## Coloration Syntaxique

Le thème utilise PrismJS pour de superbes blocs de code. Voici un exemple simple en Python :

```python
def saluer(nom):
    """Salue l'utilisateur avec un message amical de l'IA."""
    return f"Bonjour, {nom} ! Je suis votre assistant IA."

print(saluer("Développeur"))
```
